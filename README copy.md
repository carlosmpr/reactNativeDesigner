# MenuKome

 A fully serverless Smart menu for restaurants, that highlight main dishes, special of the day, and show recommendations for customers of drinks and side plates to accompany the main dish.

## Develop with
 - ReactNative.
 - AWS Lambda.
 - DynamoDB.
